Inserting Formatted Bibliographies

If you don't use LaTeX, but a wysiwyg editor such as Word, you can still use BibDesk to manage your bibliography data and insert formatted bibliographies from BibDesk in a text.

  • You can Drag and Drop or Copy and Paste items from the main Bibdesk database window. Set your Citation Preferences to get formatted bibliographies instead of LaTeX or BibTeX.
  • You can use the "Copy As" menu to copy bibliography data in many different formats, including highly customizable Templates.
  • Copy and Paste formatted text from the Previewer.
  • System Services can also be used to get formatted text, using highly customizable Templates.
