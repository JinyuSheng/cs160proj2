Scripts Menu

BibDesk has its own AppleScript menu, and scripts dedicated to BibDesk can be stored in your home directory under ~/Library/Application Support/BibDesk/Scripts.

To quickly open the scripts folder, select "About the Scripts Menu" from the "About Bibdesk Scripts" submenu in the "Scripts" menu (which is shaped like the AppleScript icon on later version of OS X). Scripts added to this folder will show up in the menu without relaunching BibDesk. You can also use the "About Bibdesk Scripts" submenu to view a sample script explaining the most important AppleScript commands you can use with BibDesk, and a sample Script Hooks handler script. To view the source of these files, select the menu item with the Option key pressed.
