Searching External Databases

With BibDesk, you can search the NCBI's Entrez set of databases, the catalogs at many research libraries using the z39.50 protocol, the ISI Web of Knowledge, and the DBLP Computer Science Bibliography.

To search these databases, select "New Search Group" from the "Searches" menu and select the desired search from the menu that appears.
