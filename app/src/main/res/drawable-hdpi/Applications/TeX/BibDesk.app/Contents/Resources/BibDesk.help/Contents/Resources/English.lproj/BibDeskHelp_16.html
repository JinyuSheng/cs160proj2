Importing References

There are many ways to add existing reference items to a BibDesk database.

  • Drag and Drop or Copy and Paste formatted text on the database window.
  • Drag and Drop or Copy and Paste formatted text files on the database window.
  • Drag and Drop or Copy and Paste files or URLs to link on the database window.
  • Drag and Drop or Copy and Paste on a detail editor window.
  • Drag and Drop, Copy and Paste, or "Merge In" items from External Groups.
  • Copy text to clipboard and choose "New Publications from Clipboard".
  • Choose "New Publications from File" to import from a file.
  • Choose "New Publications from Web" to import from a web source.
  • Select formatted text and choose "Add to Bibliography" from System Services.
  • Open other file and Drag and Drop or Copy and Paste between database windows.
  • Choose "Open Using Temporary Cite Keys" to fix bibtex with missing cite keys.
  • Choose "Open Using Filter" to preprocess data.

Here `formatted text' means bibliography data in any of a number of formats that BibDesk can recognize. This includes BibTeX, RIS, Medline (PubMed), MARC, JSTOR, and ISI Web of Science.
