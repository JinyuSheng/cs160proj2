Beta Testing

If you hold down the option key (alt on some keyboards) while the Actions menu is active, you'll see an option to Install TeX Live Manager From Critical Repository. This peculiar-sounding item allows you to install the latest TeX Live Manager test version. If you do this, please report any problems encountered to the TeX Live team via the mailing list. In addition, you can revert to a non-beta version of tlmgr by using the Reinstall TeX Live Manager menu item.
Beta versions of TeX Live Utility are occasionally available at the download site. Bugs should be reported using the issue tracker.
