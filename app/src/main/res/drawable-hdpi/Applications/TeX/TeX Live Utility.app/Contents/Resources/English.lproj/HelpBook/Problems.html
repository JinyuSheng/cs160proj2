Problems

Please report any problems using the bug tracker, or at a bare minimum, by e-mailing the maintainer at amaxwell@mac.com.
When reporting a bug, please include the following:
1. All of TeX Live Utility's log output for the problematic session
2. Exact steps to reproduce the problem
3. Expected behavior
4. Actual behavior
5. Crash logs from ~/Library/Logs/CrashReporter for any crashes.
If you are requesting a feature or asking a question, please check the FAQ first to see if it is already addressed.
