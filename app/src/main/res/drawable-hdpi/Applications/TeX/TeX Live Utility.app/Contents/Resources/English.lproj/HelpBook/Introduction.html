Introduction

TeX Live Utility is a Mac OS X graphical interface for the TeX Live Manager, a program included with recent versions of TeX Live.
TeX Live Utility features include:
Update, add, and remove packages from TeX Live.
Set default paper size for all TeX programs.
Show information and documentation for packages.
Easily switch between repositories.
Configure automatic backups in TeX Live Manager, and restore previous package versions.
Automatic checking for TeX Live updates.
Installation of TeX Live from a repository, via a graphical interface to the UNIX® install script for TeX Live.
