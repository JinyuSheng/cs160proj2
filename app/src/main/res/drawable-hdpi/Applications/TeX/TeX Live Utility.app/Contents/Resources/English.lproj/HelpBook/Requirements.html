Requirements

Mac OS X Leopard 10.5.7 or later.
TeX Live, which is commonly installed via MacTeX.
A network connection is required for updating or installing, unless you manage your own repository.
For install and update of packages, you must have write permission to the TeX Live install directory (typically /usr/local/texlive). If you are using MacTeX, this means you must be an administrator of your computer. If you installed TeX Live by some other means, or e.g., manually changed ownership of the installation directory, you may not need to be an administrator. TeX Live Utility will only prompt for an administrator password if absolutely necessary.
